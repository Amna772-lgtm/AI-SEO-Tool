**`wordpress-plugin/src/components/GeoScoreRing.jsx`:**

Pure SVG component. Re-implement GeoScoreRing with identical math to `frontend/app/components/geo/GeoScoreRing.tsx`.

Props: `score` (number 0-100), `size` (number, default 120), `grade` (string, optional)

Color function: `score >= 80 ? '#16a34a' : score >= 65 ? '#ca8a04' : score >= 50 ? '#ea580c' : '#dc2626'`

SVG structure:
- `<svg>` viewBox = `0 0 {size} {size}`
- Background circle: cx/cy = size/2, r = (size - 10) / 2, stroke `#c3c4c7`, strokeWidth 10, fill none
- Score arc: same center/radius, stroke = color from threshold, strokeWidth 10, strokeDasharray = circumference, strokeDashoffset = circumference * (1 - score/100), transform `rotate(-90 {cx} {cy})`, strokeLinecap "round", fill none
- Center text: score number, fontSize = `size * 0.24` (approx 28px at 120), fontWeight 600, fill = score color
- Sub-text: "/ 100", fontSize = `size * 0.1`, fill `#757575`, positioned below center text
- Grade badge: if `grade` prop provided, render a 32px circle below the SVG with same color as score arc, white text, fontWeight 600
