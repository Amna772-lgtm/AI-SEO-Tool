**In `backend/app/api/routes/auth.py`:**

1. Add import: `from app.store.history_store import get_subscription_by_user`
2. Add import: `from app.schemas.auth import MeOut` (alongside existing imports)

3. Define plan limits constant at module level:
```python
_PLAN_AUDIT_LIMITS = {"free": 3, "pro": 50, "agency": 200}
```

4. Modify the `/me` endpoint to return subscription data:
```python
@router.get("/me", response_model=MeOut)
def me(current_user: dict[str, Any] = Depends(get_current_user)) -> dict[str, Any]:
    sub = get_subscription_by_user(current_user["id"])
    plan = sub["plan"] if sub else "free"
    plan_status = sub["status"] if sub else "active"
    audit_count = sub["audit_count"] if sub else 0
    audit_limit = (sub.get("audit_quota_override") or _PLAN_AUDIT_LIMITS.get(plan, 3)) if sub else 3
    return {
        "id": current_user["id"],
        "email": current_user["email"],
        "name": current_user["name"],
        "is_admin": bool(current_user.get("is_admin", False)),
        "plan": plan,
        "plan_status": plan_status,
        "audit_count": audit_count,
        "audit_limit": audit_limit,
    }
```

**Create `frontend/app/dashboard/settings/api-keys/page.tsx`:**

Full API Keys management page following UI-SPEC copywriting contract. Key behaviors:

1. **State:** `keys` (list), `generating` (boolean), `keyName` (string), `newRawKey` (string | null for one-time reveal), `revokeTarget` (key object | null for confirmation), `error` (string | null)

2. **On mount:** Call `apiFetch("/auth/api-keys")` → set `keys` state

3. **Generate flow:**
   - Input field: label "Key Name", placeholder "e.g. My WordPress Site"
   - "Generate New Key" button (accent color `bg-[#0d9488]`, white text)
   - On click: POST to `apiFetch("/auth/api-key", { method: "POST", headers: {"Content-Type": "application/json"}, body: JSON.stringify({ name: keyName }) })`
   - Response: set `newRawKey` to `data.key`, refresh key list
   - One-time reveal: show key in monospace `<code>` block with copy button
   - Copy button: `navigator.clipboard.writeText(newRawKey)`, show "Copied!" for 2 seconds
   - Warning text: "Copy this key now -- it won't be shown again." in `text-[var(--error)]`

4. **Key list:**

   - Created/Last Used formatted as relative time or date

   - "Yes, Revoke" button (destructive, `bg-[var(--error)]` white text)
   - "Cancel" button
   - On confirm: DELETE to `apiFetch("/auth/api-keys/{id}", { method: "DELETE" })`, refresh list

6. **Empty state:**
   - Heading: "No API keys yet"
   - Body: "Generate a key to connect the WordPress plugin or other integrations."

7. **Page heading:** "API Keys"
8. **Page description:** "Generate API keys to connect external tools -- like the AI SEO Tool WordPress plugin -- to your account."
