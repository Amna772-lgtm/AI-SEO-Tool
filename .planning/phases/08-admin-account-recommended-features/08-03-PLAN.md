<threat_model>
| Threat | Severity | Mitigation |
|--------|----------|------------|
| Non-admin accessing admin routes | HIGH | Router-level `dependencies=[Depends(get_admin_user)]` on APIRouter constructor — applies to ALL routes |
| API key exposure in settings list | HIGH | GET /admin/system/settings masks API key values (first 8 chars + "..."); full key only via explicit reveal param |
| CSRF on destructive DELETE routes | MEDIUM | Existing SameSite=Lax cookie + CORS origin restriction; admin routes additionally require admin JWT |
| Admin deleting themselves | LOW | delete_user_cascade called with target user_id, not current user; no self-delete check needed (admin can delete self if desired) |
| Celery inspect timeout blocking API | MEDIUM | inspect(timeout=2.0) with fallback to offline state if None returned |
| Accidental job termination via cancel | MEDIUM | celery_cancel_job uses SIGTERM (graceful) not SIGKILL; frontend requires confirmation dialog before calling |
</threat_model>

    **System Health helper (per D-19, D-21, Pattern 6 & 7 in RESEARCH):**

    ```python
    def _get_system_health() -> dict:
        """Get Celery queue status and Redis memory. Non-fatal on failure."""
        celery_stats = {"active_tasks": 0, "pending_tasks": 0, "worker_online": False}
        try:
            from app.worker.celery_app import celery as celery_app
            inspector = celery_app.control.inspect(timeout=2.0)
            active = inspector.active() or {}
            reserved = inspector.reserved() or {}
            celery_stats = {
                "active_tasks": sum(len(v) for v in active.values()),
                "pending_tasks": sum(len(v) for v in reserved.values()),
                "worker_online": bool(active or reserved),
            }
        except Exception:
            pass  # Worker offline or Redis unreachable — return defaults

        redis_memory_mb = 0.0
        try:
            from app.store.crawl_store import get_redis
            r = get_redis()
            info = r.info("memory")
            redis_memory_mb = round(info["used_memory"] / (1024 * 1024), 2)
        except Exception:
            pass

        # Average audit duration from analyses table
        avg_duration = None
        failed_jobs = 0
        try:
            from app.store.history_store import _connect
            conn = _connect()
            try:
                # No explicit duration column — use analyzed_at as proxy
                # Count analyses with no score as "failed"
                row = conn.execute(
                    "SELECT COUNT(*) FROM analyses WHERE overall_score IS NULL"
                ).fetchone()
                failed_jobs = row[0] if row else 0
            finally:
                conn.close()
        except Exception:
            pass

        return {
            "celery": celery_stats,
            "redis_memory_mb": redis_memory_mb,
            "avg_audit_duration": avg_duration,
            "failed_jobs": failed_jobs,
        }
    ```

    @router.get("/system/settings")
    def get_settings():
        from app.store.history_store import get_all_admin_settings
        settings = get_all_admin_settings()
        # Mask API key values — show first 8 chars only (per RESEARCH anti-pattern)
        masked = {}
        for k, v in settings.items():

                masked[k] = v[:8] + "..." if len(v) > 8 else v
            else:
                masked[k] = v
        return masked

    @router.get("/system/settings/reveal/{key}")
    def reveal_setting(key: str):
        """Return unmasked value for a specific setting. Admin-only explicit reveal."""
        from app.store.history_store import get_admin_setting
        value = get_admin_setting(key)
        if value is None:
            raise HTTPException(status_code=404, detail="Setting not found")
        return {"key": key, "value": value}

    @router.put("/system/settings")
    def update_setting(body: UpdateSettingRequest):
